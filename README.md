# Live Chat Application

A real-time chat application built with **Next.js**, **Prisma**, **PostgreSQL**, and a custom **WebSocket server**. Project cloningan dari branch main nya cok, ini di buat broadcast jadi ga dibuat private message kalo lo buat 3 user harus nya ketiga nya dapet message yang sama. simpel nya kaya group wa, cara install nya sama kaya yang di branch main.. Inti nya beda nya sama yang branch main, ini tuh history chat gada, sama ga private message juga. db nya di pake buat nampung username sama password user doang `buat login`
